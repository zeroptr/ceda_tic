Iocomp C++ Builder 6 Installation Notes

 

Manual Package Installation

These instructions must be completed only if the automatic compile fails during the installation process...

1)     Open C++ Builder 6. 

2)     From the Main Menu, select "File->Open Project" to bring up the file open dialog.

3)     Change the "Files of Type" filter to "Builder package (*.bpk)"

4)     Browse to the "c:\iocomp\product\cbuilder6" directory (Note: Or to the alternate installation directory). 

5)     Open the "iocomp65.bpk" files. 

6)     The Package Project Manager dialog should now be open. The project is setup for all Iocomp Products by default. You may need to modify the package as specified below based on which Iocomp Products you have purchased.

Without Std Pack (Included in Pro and Ultra Pack)
Remove iInstrumentPackPropertyEditorsReg.pas 
Remove iInstrumentPackReg.dcr 
Remove iInstrumentPackReg.pas

Without Pro Pack (Pro Pack included in Ultra Pack)
Remove iProfessionalPropertyEditorsReg.pas 
Remove iProfessionalReg.dcr 
Remove iProfessionalReg.pas

Without Plot Pack(Plot Pack included in Ultra Pack)
Remove iPlotPackPropertyEditorsReg.pas 
Remove iPlotPackReg.dcr 
Remove iPlotPackReg.pas

Without OPC Pack
iOPCCommonPropertyEditorsReg.pas

iOPCCommonReg.dcr

iOPCCommonReg.pas

Without Ultra Pack
Remove iInstrumentPackPropertyEditorsReg.pas 
Remove iInstrumentPackReg.dcr 
Remove iInstrumentPackReg.pas 
Remove iProfessionalPropertyEditorsReg.pas 
Remove iProfessionalReg.dcr 
Remove iProfessionalReg.pas

Remove iPlotPackPropertyEditorsReg.pas 
Remove iPlotPackReg.dcr 
Remove iPlotPackReg.pas

7)     From the Project Manager, click the Install button.

8)     If you receive a dialog warning about the forms needed to be closed, click "Yes" as the response. 

9)     You will now receive an information dialog listing the new controls being added to the palette, click the "OK" button.

10) Close the Iocomp65.bpk Package Manager and click YES when prompted to save changes. 

11) From the Main Menu, select Tools->Environment Options. 

12) Select the Library tab. 

13) Click on the Library Path browse button ("...") to modify it. 

14) In the edit-box, below the list box, enter the path to the Iocomp source code ("c:\iocomp\product\cbuilder6") and then click the "Add" button. 

15) Click the "OK" button to close the "Directories" dialog. 

16) Click the "OK" button to close the "Environment Options" dialog. 

17) You are now ready to start using Iocomp Instrumentation Components.
