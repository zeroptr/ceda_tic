Compile Service.dpr project, open the folder which contains the compiled file and run INSTALL.BAT to install this new NT Service or UNINSTALL.BAT to uninstall it.
Then, go to "Control Panel/Administration Tools/Service" and start "EurekaLog 6.x - Service Demo" service.

Note: This project uses the Dll.Dll library to show EurekaLog ability to intercept DLL exceptions.
      You can find DLL source code in ...\Demos\DLL directory.
